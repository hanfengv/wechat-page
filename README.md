# wechat-page README

## 功能

1. 打开同一文件夹下的文件. 比如:微信小程序页面的相关文件;
2. 在 `.vscode/template/` 目录下定义模板文件夹, 在编辑器左侧 右击文件夹 选择菜单 `CopyTplHere`

## 命令

```js
"command": "wechat-page.wePage",
"title": "wechat-page[wePage]: 显示与当前文件关联的 page"
```
